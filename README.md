# 🌿 Ruh-Care

**Professional Cupping Therapy & Wellness Management Platform**

Ruh-Care is a specialized mobile application designed to bridge the gap between traditional healing practices and modern digital convenience. Built for a cupping therapy (Hijama) clinic, it serves patients, doctors, and administrators with a seamless, user-centric experience.

---

## ✨ Features

### 🏥 For Patients
- **Therapy Discovery**: Browse specialized treatments like Hijama, Fire Cupping, and Leech Therapy with detailed benefits.
- **Instant Booking**: Real-time appointment scheduling with automated slot management.
- **Wellness Store**: Shop for premium natural products like Shilajit, Talbina, and specialized oils.
- **Secure Auth**: Role-based login and profile management.

### 🩺 For Doctors
- **Daily Schedule**: Real-time view of upcoming patient sessions.
- **Patient Management**: Track therapy history and session completion.

### ⚙️ For Administrators
- **Inventory Tracking**: Manage clinic supplies and store stock levels.
- **Operational Overview**: Monitor all bookings and orders from a central dashboard.

---

## 🚀 Tech Stack

- **Frontend**: Flutter (Cross-platform excellence)
- **Backend**: Firebase (Serverless Scalability)
  - **Authentication**: Firebase Auth
  - **Database**: Cloud Firestore (NoSQL)
  - **Storage**: Firebase Storage (Media & Assets)
  - **Logic**: Cloud Functions (Secure dynamic processing)

## 🛠️ Getting Started

### Prerequisites
- Flutter SDK (Latest Stable)
- Android Studio / VS Code
- Firebase Project configured

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Rizwan027/Ruh-Care.git
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
